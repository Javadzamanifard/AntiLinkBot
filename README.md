 🤖 ربات ضد لینک تلگرام (Anti-Link Bot)

یک ربات تلگرام ساده و کاربردی برای حذف لینک‌ها در گروه‌ها و کانال‌ها.  
ساخته شده با **Python** و کتابخانه‌ی قدرتمند [PyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI).

---

## 🚀 ویژگی‌ها

- 🔗 **حذف خودکار لینک‌ها** در گروه‌ها (t.me، telegram.me، http، https)
- 👮‍♂️ **پنل مدیریت مخصوص ادمین**
- 🟢 **فعال / غیرفعال کردن ضد لینک**
- 🧑‍🤝‍🧑 **مدیریت لیست سفید** (کاربرانی که از حذف لینک معاف هستند)
- 💾 **ذخیره تنظیمات در فایل JSON** (عدم نیاز به دیتابیس)
- ⚙️ **دکمه‌های تعاملی** برای تنظیم سریع از داخل تلگرام

---

## 🛠️ پیش‌نیازها

برای اجرای پروژه باید موارد زیر نصب باشند:

- Python 3.8+
- کتابخانه‌ی PyTelegramBotAPI

نصب کتابخانه با pip:

```bash
pip install pytelegrambotapi
📦 ساختار پروژه
bash
Copy code
anti_link_bot/
│
├── bot.py              # فایل اصلی ربات
├── settings.json       # فایل ذخیره تنظیمات (خودکار ساخته می‌شود)

⚙️ راه‌اندازی
توکن ربات خودت رو از BotFather بگیر.

مقدار TOKEN و ADMIN_ID را در فایل bot.py تنظیم کن:

python
Copy code
TOKEN = "YOUR_BOT_TOKEN"
ADMIN_ID = 123456789  # آی‌دی عددی ادمین
اجرای ربات:

bash
Copy code
python bot.py
حالا ربات آماده است!
ادمین می‌تواند با دستور /start وارد پنل مدیریت شود و تنظیمات را تغییر دهد.

🧩 دستورات اصلی
دستور / دکمه	توضیح
/start	شروع ربات و نمایش پنل
👮‍♂️ پنل ادمین	ورود به بخش مدیریت
⚙️ تنظیمات ربات	فعال/غیرفعال کردن ضد لینک
➕ افزودن کاربر به لیست سفید	کاربر را از حذف لینک معاف می‌کند
➖ حذف کاربر از لیست سفید	حذف از لیست سفید
🔙 بازگشت	بازگشت به منوی قبلی

🧠 نکات
فقط کاربری که آی‌دی‌اش در ADMIN_ID تعریف شده به پنل دسترسی دارد.

همه تنظیمات (لیست سفید و وضعیت ضد لینک) در فایل settings.json ذخیره می‌شوند.

ربات قابلیت اجرا روی هاست یا سرور دائمی (مثل VPS یا Heroku) را دارد.
